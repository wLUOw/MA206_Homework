\section{第1题}
根据万有引力公式
\[\frac{Gm}{r^2} = \omega^2 r\]
以及
\[\omega = \frac{2\pi}{t}\]
可以解出
\[t = \frac{2\pi}{\sqrt{G}} m^{-\frac{1}{2}} r^\frac{3}{2}\]
因此可以得到$a = -\frac{1}{2}$、$b = \frac{3}{2}$和$c = -\frac{1}{2}$
